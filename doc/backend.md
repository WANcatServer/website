# 後端

* 首頁 — Cookie 檢查登入，初次進入 Cookie 聲明
* 登入 — 登入成功 301 Dashboard
* Dashboard — 推薦題目、我的題目、最近活動（自己）、分類
  * 推薦題目：偏好及推薦系統。
  * 最近活動：Record 中 user == $user
* 推薦系統 函數：時間、作答人數、評比
* 題目網址 /\$username/\$question-name：
  * /
  * /answering   //答題頁面
  * /show-answers   //答題後頁面
  * /forum
  * /reference
  * /knowledgebase
  * /issue
